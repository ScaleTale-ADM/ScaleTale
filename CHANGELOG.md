# Changelog

All notable ScaleTale changes will be documented in this file.

---

## V71 - Music & SFX Beta

### Added

- Dynamic music system by game context.
- Organized asset folder support.
- New soundtrack routing for menu, planets, Scale-0, cockpit, bosses and loot.
- Sound effects system with controlled volume and cooldowns.
- SFX for player shots, laser charge, dash, cockpit scan, boss intro, boss charge, shield impact, pickups, coins, loot and explosions.

### Improved

- Asset loading now searches inside the organized `assets` folder.
- Music and SFX are separated into clearer folders.
- Audio is less chaotic by using cooldowns for repeated effects.

---

## V70 - Asset Integration & Boss Loot Beta

### Added

- Integrated visual asset support for planet hazards.
- Boss loot system after defeating bosses.
- Loot cards with visual icons.
- Cockpit scanner assets.

### Improved

- Boss reward flow.
- Visual presentation of some planet-specific effects.

---

## Previous Versions

Older versions include experiments and beta work on:

- Scale-0 mystery event.
- Top-down Scale-0 labyrinth.
- Planet selector.
- Planet atmospheres.
- Cockpit boss intro system.
- Gameplay expansion systems.
- Boss polish and visual improvements.
