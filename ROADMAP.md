# ScaleTale Roadmap

This roadmap shows possible future directions for ScaleTale. It may change as the project evolves.

---

## Short Term

- Stabilize V71 audio systems.
- Improve organized asset loading.
- Prepare the first public GitHub release.
- Improve documentation for players.
- Create a cleaner download package.
- Add Discord feedback workflow.

---

## Medium Term

- Improve Scale-0 exploration.
- Add deeper top-down gameplay for the pilot.
- Expand planet-specific gameplay.
- Improve boss polish and cinematic attacks.
- Add more clear progression systems.
- Improve performance and code structure.

---

## Long Term

- Prepare a standalone `.exe` build.
- Improve modding support.
- Add more professional menus and onboarding.
- Explore itch.io or Steam-style distribution.
- Build a stronger community testing process.
- Continue improving visuals, audio and gameplay depth.
