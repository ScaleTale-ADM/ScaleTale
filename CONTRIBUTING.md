# Contributing To ScaleTale

Thank you for wanting to help improve ScaleTale.

ScaleTale is a personal indie project, and contributions are welcome when they help make the game more stable, fun, clear or visually polished.

---

## English

### Ways To Contribute

You can help by:

- reporting bugs
- testing beta versions
- suggesting gameplay improvements
- suggesting boss or planet balance changes
- creating free mods
- improving documentation
- improving code readability
- proposing visual or audio improvements

### Contribution Rules

- Be respectful.
- Explain changes clearly.
- Do not submit malware, cheats or harmful code.
- Do not submit paid unofficial versions.
- Do not claim unofficial versions are official.
- Keep changes focused and easy to review.

### Bug Reports

Please include:

- game version
- operating system
- what happened
- what should have happened
- steps to reproduce
- screenshot or error message if possible

### Modded Versions

Modded versions are allowed if they are free and clearly marked as unofficial.

Do not sell modified versions of ScaleTale without explicit permission.

## Espanol

### Formas De Contribuir

Puedes ayudar:

- reportando bugs
- probando versiones beta
- sugiriendo mejoras de gameplay
- sugiriendo cambios de balance en bosses o planetas
- creando mods gratuitos
- mejorando la documentacion

- proponiendo mejoras visuales o de audio

### Reglas Para Contribuir

- Se respetuoso.
- Explica los cambios con claridad.
- No envies malware, trampas o codigo danino.
- No envies versiones no oficiales de pago.
- No digas que una version no oficial es oficial.
- Mantén los cambios centrados y faciles de revisar.

### Reportes De Bugs

Incluye:

- version del juego
- sistema operativo
- que ocurrio
- que deberia haber ocurrido
- pasos para reproducirlo
- captura o mensaje de error si es posible

### Versiones Modificadas

Las versiones modificadas estan permitidas si son gratuitas y estan claramente marcadas como no oficiales.

No vendas versiones modificadas de ScaleTale sin permiso explicito.
