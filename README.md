# ScaleTale

**ScaleTale** is a free Python/Pygame space action game created as a personal indie project.

**Current public beta:** `V71 - Music & SFX Beta`

ScaleTale combines arcade space combat, boss fights, planet selection, Scale-0 mystery events, cockpit scanning, dynamic music, sound effects and boss loot rewards.

---

## English

### Features

- Arcade space combat made with Python and Pygame.
- Multiple enemy types and cinematic boss fights.
- Planet selector with different visual atmospheres.
- Scale-0 mystery content and top-down exploration.
- Cockpit scan system before major boss fights.
- Dynamic background music by game context.
- Sound effects for weapons, bosses, cockpit, loot, impacts and pickups.
- Boss loot system with temporary upgrades and rewards.
- Spanish and English language support.

### How to Play

1. Download the latest release from GitHub.
2. Install Python 3.
3. Install Pygame:

```bash
pip install pygame
```

4. Run the main Python file:

```bash
python SCATEL_V71_MUSIC_ASSET_STRUCTURE_BETA.py
```

### Recommended Download

For normal players, use the latest `.zip` from the GitHub Releases section. It should include:

- the main `.py` file
- the `assets` folder
- documentation
